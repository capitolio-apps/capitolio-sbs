## Phase Details

### Phase 1: Complete Website

**Goal:** Fully functional minimalist portfolio website deployed and ready for lead capture

**Depends on:** Nothing (first phase)

**Requirements:** All v1 requirements (DESN-01 through TECH-04)

**Success Criteria** (what must be TRUE):

1. Landing page displays with John Pawson minimalist aesthetic — whitespace over borders, monochromatic palette, high-contrast typography
2. Light mode toggles to dark mode (charcoal/grey tones) and back, with smooth subtle transition, and preference persists across browser sessions
3. Language toggles between English and German, all visible text is translated, and preference persists across sessions
4. Three-section content architecture loads: CTA section (booking path), Proof of Work section (3 achievement cards), Pricing section (3.600€/month)
5. Privacy Policy, Accessibility Statement, and Impressum pages are accessible and display exact legacy content
6. Site is responsive on mobile, tablet, and desktop with semantic HTML structure
7. Static site is deployable to GitHub Pages with vanilla HTML/CSS/JS (no frameworks)

**Plans:** 3 plans

- [ ] 01-foundation-PLAN.md — Design system CSS and JS utilities (theming, i18n)
- [ ] 02-content-PLAN.md — Landing pages with CTA, Proof of Work, Pricing sections
- [ ] 03-legal-PLAN.md — Privacy Policy and Accessibility Statement pages
